Now that you've seen how fast and easy file i/o can be in Python, let's take a look at one of my favorite libraries that helps us do even more. This library is called "Pandas". Pandas is great at storing a lot of data with complicated internal structures. This library is also awesome because of how smoothly it can write data to nice formats! Check out this example of a file that crawls our current directory (wherever the file_summarizer.py file is located) and collects the name of each file, then writes an Excel summary of the information.

Ok, now let's break the file down line by line.
The "os" module comes with Python itself. It contains functions to deal with the operating system. We'll use it to collect the files in the same directory as this script.


The "pandas" module is not built-in, it's a third-party module. It's great for handling lots of data and exporting it to useful formats.

Make a list that will store the names of all the files in this folder.

Then use the os library's 'listdir' method to get the names of all the files in the directory '.'.
In the command line of the computer, the '.' directory means "whatever directory the script is currently inside."

Lastly, put our data into a Pandas data structure called a "DataFrame." This type is specific to Pandas, and you can read more about it at this link. A DataFrame in Pandas is very easy to write to Excel. Just use the dedicated 'to_excel' function.
