Variable Scope

Now let's talk about the scope of a variable. The scope of a variable is sort of like its territory. Inside the scope, the variable has meaning and can be called. Outside of the scope, the variable is not defined and meaningless. It will cause an error.

An Unknown Number of Parameters

Python has some neat syntax that lets us pass in an unknown number of parameters. We use *args to represent some arguments, and they can change without rewriting the function! Pretty neat.
