if/else structure

  • a colon goes after the if statement, elif statement, and else statements
  • blocks with actions that follow these must be indented consistently
  • if and elif (else if) contain a statement that is either True or False
if test1:
	statements1
elif test2:
	statements2
else:
	statements3
