While loops

A while loop is different from a for loop because it will continue to loop until some condition is met. There can be an indefinite number of repetitions.

Control Flow with while

  • break will break out of the loop and go to the next statement outside of the loop without executing anything else within the loop.
  • continue will return to the top of the loop without completing the rest of the code in that block this time through.
