Terminology

Defining a function means that you are making your tool. Calling a function means you are ready to use it. Parameters are the variables that you specify for your function, for example things you give to your scissors, or advertiser. Return is what they give back to you. Function Notes

  • Functions involve stand alone definitions AND calls to a function within a program or several programs. (mention turtle functions)
  • A function is defined using def, the name you choose, and () with or without parameters for input.
  • The function may or may not end with a return statement.
  • The functions would precede the main program if you have them in one file.
  • Variables that you define as parameters for your function don’t have to have the same name as the parameters that you send to the function. Variables in our examples are matched by position.
