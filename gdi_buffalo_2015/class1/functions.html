Functions

Terminology

Defining a function means that you are making your own tool. You make up a name for the function. The name is like a variable, but it has parentheses after it, with or without parameters. Calling a function means you are ready to use it. Parameters are the variables that you specify for your function, for example the length of a side of a square. Return tells what the function gives back to you. There doesn't necessarily have to be parameters or returns.

Function Notes

  • Functions involve stand alone definitions AND calls to the function within a program. The definition must come before the call.
  • A function is defined using def, the name you choose, and () with or without parameters for input.
  • The function may or may not end with a return statement.
  • Variables that you define as parameters for your function don’t have to have the same name as the parameters that you send to the function. Variables in our examples are matched by position.
