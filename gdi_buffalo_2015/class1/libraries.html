Libraries, Modules, Packages

libraries are essentially sets of functions that other people have written and that we might want to make use of. Turtle is an example of a library. functions like these are contained it it:
  • forward(number_of_steps)
  • left(number_of_degrees)
  • Screen()
  • Turtle()

Options for getting and using these turtle functions

Reserve words include import, as, from Convention is to put all import statements at the top of the file. This example uses as to give turtle a nickname. I can use from to import specific functions from the turtle library. Now I don't have to use the dot notation for all.
