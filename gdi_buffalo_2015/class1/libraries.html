Libraries, Modules, Packages

libraries are essentially sets of functions that other people have written and that we might want to make use of. Turtle is an example of a library. functions like these are contained it it:
  • forward(number_of_steps)
  • left(number_of_degrees)
  • Screen()
  • Turtle()

Options for getting and using these turtle functions

Related commands, or reserve words, include:
import
as
from
*
Look over this example. Go ahead and try it if you don't remember how it worked. A convention in programming is to put all import statements at the top of the file. This example uses the as command to give turtle a nickname. Read it over. Trying is optional. I can use the from command to import specific functions from the turtle library. Notice how now I don't have to use the dot notation for all. Here I am importing all functions from turtle and again I don't need to use dot notation.
